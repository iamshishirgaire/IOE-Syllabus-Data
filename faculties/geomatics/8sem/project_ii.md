# Syllabus

**PROJECT WORK II**

**Course Objective:**  
Project Work II is the continuation of Project Work I. In Project II students are required to complete following works in carry-over of Project I falling under different categories of project works:

1. **Design type project**
2. **Dissertation type project**
3. **Experimental type project**

In the initial phase the faculty may conduct a number of lectures and discussions as to the approach of the project. In the later phase the student will be left on his own to pursue his work and to consult the faculty whenever any problem crops up.
